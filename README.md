# rogilinkFlex-esp32

現在開発中の汎用通信ライブラリrogiLinkFlexのESP32実装です。
動作未確認です。UART通信のみに対応しています。従来のrogilinkに比べて、送受信するデータ型を柔軟に設定できるようになっています。マイコン同士の通信、マイコンとPCの通信にも使えます。
